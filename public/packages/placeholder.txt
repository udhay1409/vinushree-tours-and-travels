# Package Images Directory

This directory contains images for tour packages.

## Structure:
- `/packages/` - Main package images
- `/packages/gallery/` - Gallery images for packages

## Image Guidelines:
- Main images: 800x600px recommended
- Gallery images: 600x400px recommended
- Formats: JPG, PNG, WebP
- Max size: 5MB per image
